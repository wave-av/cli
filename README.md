# @wave-av/cli

Command-line interface for the WAVE streaming platform. Manage streams, productions, and infrastructure from your terminal.

## Installation

```bash
npm install -g @wave-av/cli
npx @wave-av/cli stream list
```

## Quick start

```bash
wave login
wave stream create --title "My Stream" --protocol webrtc
wave stream list
wave stream start <stream-id>
```

34 command groups covering streams, studio, clips, voice, phone, captions, chapters, AI, fleet, analytics, billing, and more.

See the [full documentation](https://www.npmjs.com/package/@wave-av/cli) for all commands, authentication options, and CI/CD examples.
